Orbital Horizon

Project Repository Link

Orbital horizon is a personal project I am creating with a friend. Its about a person who is sent into space to farm and survive as long as they can whilst being forced to make decisions to choose which space politics to join. The project started in the start of summer 2024, and is still in development. We plan on this project to be a fully fully released this game on steam in the future. I am the lead programmer, who also does a lot of game designing and direction of the game. I am also doing pixel art and music.

Tools used are using unity, jira, confluence, github, unity, fl studio, aseprite and csharp.

Cave Generation

linkedIn project post (more about this)

Project Repository Link

This is a project created in unity with the intended prupose of creating minecraft-like generated caves within a 3D space. This was a a fun 2 person project I did with a classmate. We both did a lot of research as we had no idea how to accomplish this. We both did a lot coding and long pair coding hours. We learned a lot in this! In our project, we created procedural world generation using 3D Perlin noise. We also created chunking with the help of spatial quantisation.

Tools used were using github, unity, and csharp.
